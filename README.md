# 人要有敬畏之心  

## 致敬那些无以名状之存在 

&nbsp;

我以前认为创作是一件危险的事情  

现在我不这样认为了

&nbsp;

但是  

我始终觉得  

人要有敬畏之心
&nbsp;

我们老家有句老话是说  

举头三尺有神明

&nbsp;

我觉得写的时候  

人要对自己诚实  

非说不可的才要写下来

&nbsp;

其他的都是随缘了  
&nbsp;

我给自己写的东西取名叫  

把话说给宇宙听  

其实也是对无以名状之存在的敬畏
